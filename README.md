# JavaScript Practice Projects

This repository contains JavaScript practice exercises and small web projects created while learning front-end web development.

The goal of these projects was to understand how JavaScript interacts with HTML and CSS to build dynamic and interactive web pages.

--------------------------------------------------

TOPICS COVERED

DOM Manipulation
Examples showing how JavaScript can modify HTML elements dynamically.

Event Handling
Projects demonstrating click events, hover events, and user interaction.

Functions
Practice using JavaScript functions to organize code and perform tasks.

Query Selectors
Examples using querySelector and querySelectorAll to access and modify elements.

User Input Handling
Projects demonstrating how to read and respond to user input.

--------------------------------------------------

PROJECTS INCLUDED

Profile Page
A simple web page demonstrating layout and interactive elements using JavaScript.

Query Selector
Examples showing how to select and modify HTML elements using JavaScript.

Dojo Sports
A small web project demonstrating user interaction and page updates.

Likes
Examples demonstrating button clicks and like counters.

Video Preview
A project demonstrating video preview functionality using JavaScript.

Hover Effects
Examples showing how hover events can change page behavior.

--------------------------------------------------

TECHNOLOGIES USED

JavaScript  
HTML  
CSS  

--------------------------------------------------

GOAL

The goal of this repository is to practice front-end development concepts and understand how JavaScript interacts with the browser.
